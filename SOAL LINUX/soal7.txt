Buat direktori bernama backup di dalam direktori /opt.
jawab : mkdir /opt/backup

Salin seluruh isi dari direktori /etc ke dalam direktori /opt/backup.
jawab : cp /etc /opt/backup

Ganti nama file log.txt menjadi log_baru.txt.
jawab : mv log.txt log_baru.txt

Hapus direktori /opt/backup beserta seluruh isinya.
jawab : rm -rf /opt/backup

Buat file baru bernama data.log di dalam direktori /var/log.
jawab : touch /var/log/data.log

Ubah permission file data.log agar hanya bisa dibaca oleh semua user.
jawab : chmod 400 data.log

Tampilkan ukuran file data.log dalam format yang mudah dibaca.
jawab : du -h /var/log/data.log

Pindahkan semua file .log dari direktori /var/log ke /tmp.
jawab : mv /var/log /tmp
